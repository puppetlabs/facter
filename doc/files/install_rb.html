Install file(s) from ./bin to RbConfig::CONFIG[‘bindir’]. Patch it on the way to insert a #! line; on a Unix install, the command is named as expected (e.g., bin/rdoc becomes rdoc); the shebang line handles running it. Under windows, we add an ’.rb’ extension and let file associations do their stuff.
